\subsection{Definitions}
$$\lim_{x\rightarrow c} f(x) = L$$
L is the limit of f(x) as x approaches c, if for every $\epsilon > 0$ there exists a $\delta > 0$ such that if $0 < |x - c| < \delta$ then $|f(x) - L| < \epsilon$.
\\
\noindent if f is continuous at c then $\lim_{x\rightarrow c} f(x) = f(c)$
